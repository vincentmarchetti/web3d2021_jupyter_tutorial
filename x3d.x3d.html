x3d package is a binding to the X3D specification maintained by the Web3D Consortium. It is distribute on PyPI at https://pypi.org/project/x3d/

Importing x3d package

As a result of the way the x3d package is distributed from PyPI, an import statement:

import x3d
will not import any classes into the global namespace.

The alternative forms to use are:

from x3d import *
whichl will import classes and methods into the local namespace.

from x3d import x3d

Will import classes and methods into the local namespace packaged inside a module x3d.
